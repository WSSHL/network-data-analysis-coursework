## How to run

Put `CW_CODE.ipynb` and the `datasets/` folder in the same parent folder. In other words, the notebook and the datasets folder should sit side by side, for example:

```text
project_folder/
├── CW_CODE.ipynb
└── datasets/
```

Then:

1. Create a Python environment with the required packages.
2. Open `CW_CODE.ipynb` in Jupyter Notebook or JupyterLab.
3. Run the notebook cells from top to bottom.
